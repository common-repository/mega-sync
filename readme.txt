== Description ==

This plugin enables HubSpot and Salesforce forms integration with Contact Form 7 forms and allow user to specify SMTP settings that can be used to email form.
Once installed and activated:-

	=>Each CF7 form will have a separate settings section where you can add HubSpot form credentials. i.e, Portal ID, Form ID, Form Fields, etc. When the plugin is activated you will have the option to enable/disable the form integration. In enabled mode, the status code and response from HubSpot will be stored for the last form submission so you can debug if there are any issues.
	
	=>Each CF7 form will have a separate settings section where you can add Salesforce form credentials. i.e, Salesforce URL, Salesforce ID, Form Fields, etc. When the plugin is activated you will have the option to enable/disable the form integration. In enabled mode, the status code and response from Salesforce will be stored for the last form submission so you can debug if there are any issues.
	
	=>It will also allow users to change SMTP settings that can be used to email the form.
